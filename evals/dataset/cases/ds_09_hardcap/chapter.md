# 第二十章 换道

后晌收工,等人都散了,沈砚才摸到老陈歇脚的窝棚。老陈正就着咸菜啃饼,见他来,眼皮一抬:"想通了?"

沈砚把一小串钱搁在木墩上,又添了那把削矿的好刀:"夜里子时到寅时,3号矿道那条空道,归我一个人走。名册上别记我的名。"

老陈捏起那把刀,借着火光看了看刃口,又掂了掂钱串,脸上的褶子松开些:"刀是把好刀。"他把钱和刀都拢进怀里,"成。子时到寅时,那道我给你空出来,名册上你那一栏,我填个病歇。"

沈砚要的不是这几两钱能省下的工,而是一条没人盯着、能自由进出的活路。

他点点头,又把话钉死:"说定了,今夜就起。你要是漏了嘴,咱俩谁都落不着好。"

老陈嘬了口咸菜:"放心,我这张嘴,收了钱就比夜道还严。"

沈砚这才转身出棚。夜风一吹,他摸了摸空了的钱袋,心里却比来时踏实——从今夜起,那条道是他的了,退路又宽出一截。
