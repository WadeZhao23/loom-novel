# 第十三章 循灯

天一亮,沈砚就没急着往矿道去。他借着换班的空当,顺着提灯没入的那条巷道摸回去,一寸一寸看地上的痕迹。

湿泥地上果然留着脚印,鞋底沾过桐油,踩一步印一圈油光,一直往废料场的方向去。沈砚蹲下身,指尖蹭了蹭那点油渍,还没干透。

他心里翻来覆去过着前日夜里的动静——那盏提灯、那道喊破他真名的声音,像根刺扎在心口,拔也拔不出来。

沈砚把脚印的方向、鞋底的油、声音的口音一样样记下。喊他名字的人,咬字带着城里的腔,矿场里少有这样的口音,是外头来的。

顺着油痕,他在废料场的断墙后头翻出半截烧剩的灯芯,还有一枚铜制的灯钩,钩上刻着极小的一个记号。

那记号眼熟。他盯着看了半晌,认了出来——是坊市信局替人递信才用的火漆钩。这么说,喊破他真名的,多半是受人所托来探底的跑腿,身后另有主家。

沈砚把灯钩揣进袖里,心里稍稍定了些。对方既是来探底,就还没把他一口咬死;他还有工夫先摸清主家,再定走留。他不动声色回了矿道,像什么都没发生过。
