# 第十二章 过磅

天刚亮,矿场的过磅棚就排起了队。沈砚拎着自己那袋矿砂,不声不响排在最后。

赵矿头坐在磅秤后头,眼皮都不抬。轮到沈砚,他掂了掂,随口报:"三成七,记账。"

沈砚没接他的话,只把袋口解开,从里头摸出一张折得整齐的纸,平铺到磅秤边上。纸上是半个月来每一趟出砂的斤两,一笔一笔,连日子都对着矿灯编号。

"上月同样的砂,你给的是四成五。"他声音不高,"这半个月十七趟,按三成七结,少的数都在这纸上了。"

赵矿头的手停在算盘上。棚子里排队的矿工不吭声,眼睛却都往这边瞟。

沈砚不催,也不多说一个字,就那么站着,等他自己算。

算盘珠子响了一阵,赵矿头脸上青一阵白一阵,末了把那三成七划掉,重重写下四成五:"……记你四成五。今天的,补齐。"

沈砚收起纸,道了声谢,转身走了。身后有相熟的老矿工低声笑:这油盐不进的赵矿头,头一回吐出去这么多。

出了棚子,沈砚把那张纸重新折好收进怀里。补回来的分成够他多囤半月的口粮——在这矿场,多一分家底,就多一分退路。他没露声色,脚步却比来时轻了些。
