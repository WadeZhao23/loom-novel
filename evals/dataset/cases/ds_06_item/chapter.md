# 第七章 余震

头顶那道裂缝还在渗石子,簌簌地往下掉。沈砚一把攥住周楠的手腕,把她往最近的支撑木下拽:"别动,等它过去。"

第二波余震来的时候,整条矿道都在闷响。碎石砸在两人蜷着的背上,沈砚拿肩膀替她挡了大半。也不知过了多久,头顶的响动才一点点弱下去,尘土像雾一样悬在火光里,呛得人睁不开眼。

"没事了。"沈砚先探出头,看清退路没塌,才把周楠拉起来。她额角被划了道口子,渗着血。

沈砚从怀里掏出那瓶金创药,拧开塞子,倒出一撮,先给她敷上,又在自己被碎石划破的臂上按了些。药粉沾着血,凉丝丝地渗进去。

尘埃落定,他扶周楠在干处坐下,就着火光探退路。三号道被这一震塌了半边,原先的出口走不通了。

周楠喘匀了气,压低声音:"往里走,过了那口枯井,有条老排水道,是早年泄矿水挖的,窄,但能爬出矿场。矿上没几个人知道。"

沈砚把这条道记下,心里那根绷了半宿的弦松了半分。明路塌了,好歹还剩条暗路。他把周楠护在身后,举着火,一步一探往枯井那头摸去。
