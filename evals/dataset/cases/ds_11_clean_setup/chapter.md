# 第十八章 陪笑

清脉司的王主事进场那日,沈砚破天荒凑了上去。

他平日闷得像块矿石,这天却提了一壶热茶,笑着挤到王主事跟前:"主事一路辛苦,矿上没好东西,先润润喉。"

四下的矿工都看愣了——那个见人不搭话的沈砚,今儿怎么这般会来事。

王主事斜他一眼,倒没推,接过茶盏呷了一口。沈砚就势陪在旁边,一句一句捧着,眼睛却没歇。他看王主事怎么翻名册:一页一页照矿道编号排,验一个,朱笔勾一个,勾过的翻到后头,没勾的压在最上。

"主事这查法,真是滴水不漏。"沈砚陪着笑,心里把那本名册的记法记了个透。要过明日的查,得赶在朱笔勾到自己那页之前,把顺序动一动。

王主事被捧得受用,松了口:"明日辰时开验,三号矿道头一个。手脚干净的,过得快。"

沈砚心里一沉——三号矿道,正是他。面上却半分不显,又替他续上茶:"那小的们今夜可得把自个儿拾掇利索。"

送走王主事,沈砚立在原地,慢慢收了脸上的笑。

这一晚陪笑没白搭。几时开验、谁排头一个、名册怎么记,他都摸清了。硬闯是死路,可这册子既按矿道编号排,就有能钻的空子。他不急着动,先把每个环节在心里过了一遍:今夜先探清管册的人几时换岗,明日辰时之前,总寻得着那一线生机。

他转身回棚,脚步比来时沉,心里却亮了一块。
