# 第十八章 帐中话

天没亮透,矿道最深处只有一点松脂火。周楠猫着腰钻进来,肩上还沾着夜里的湿气,声音压得极低:"谷口那队人扎了帐,帐门落了锁,守得铁桶一样。我探了半宿,连帐边都近不得,不知道他们在合计什么。"

沈砚靠着支撑木坐着,火光在他脸上跳。他闭了闭眼,忽然开口,像在念一段听得清清楚楚的话:

"此刻那帐里,为首的正对副手说——『明日辰时收网,先拿三号矿道那个眼底泛青的,验明就地锁了。』副手应了声『是』,又添一句:『其余矿工不必惊动,免得走了风。』"

周楠猛地噎住,火折子的光晃了一下。"辰时收网……"她喃喃,顾不上别的,先掐着指头算时辰,"从现在起,不到两个时辰。"

沈砚睁开眼,盯着火。"眼底泛青的,这矿场只我一个。"他说得很平,"他们要的是我。"

他没再多解释一个字,只把话头拨向退路。"三号道通废弃排水暗渠,我早探过,能出矿。"沈砚站起身,把松脂火按灭,"你走明面,天亮照旧出诊,别沾上我。我走暗渠,趁他们收网之前先出谷。"

周楠咬着唇,从药箱底摸出一小包干粮塞给他:"路上吃。"

沈砚接了,掂了掂那点分量,头一回觉得这矿场里还有人肯替他兜半分风险。他把干粮塞进怀里,退进更深的暗处,摸黑往三号道去。身后,谷口的方向还是死寂——那帐门始终锁着,没人进,也没人出。
