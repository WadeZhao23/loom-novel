# 第十六章 夜遁

天擦黑,矿工棚的油灯一盏接一盏灭了。沈砚就着最后一点天光,把周楠那张草图摊在膝头,一条一条暗渠的走向记进心里。

在这个世界上,每个人都背着自己的秘密,而沈砚背负的秘密,比任何人都要沉重。

他把值钱的细软一样样收进包袱:半吊钱、一把矿刀、周楠给的伤药、还有那张草图。收拾停当,他的心中充满了复杂的情绪,既有对未知前路的恐惧,也有对这片矿场的不舍与留恋。

他知道,自己不仅要躲开清脉司的耳目,而且要在天亮前赶到谷口,因此他必须争分夺秒,不能有半分迟疑。

沈砚把油灯吹熄,借着记熟的暗渠走向,猫着腰摸出矿工棚。头一段暗渠又低又窄,他半跪着往前挪,草图上标的三处岔口,一处不差。

摸到第二处岔口,他停了停,按草图拐向左边那条老河沟——这条通谷外的道,是周楠特意圈了红的。渠壁渗水冰凉,他一步一探,总算没走错。

后半夜,沈砚从河沟尽头钻出来,矿场的火把已落在身后老远。他回头看了一眼那片黑黢黢的矿架,把草图就着河水泡烂、揉碎,撒进水里。逃出这一步,总算替自己挣回了一线活路。
