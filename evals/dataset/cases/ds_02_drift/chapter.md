# 第九章 钟响

铜钟还在震,余音撞得矿道嗡嗡作响。沈砚攥着矿牌冲到井口,先没露头,贴着支撑木往外看。

矿场晒谷坪上落了三个人,黑衣束袖。围拢的矿工被他们随手一挥,像拨麦子一样拨开。

沈砚没动。他先数人头,再看落脚——这是他这三年养出的规矩,险境里先弄清对面几斤几两,再想退路。

为首那人足尖在矿壁上虚点一下,整个人凌空拔起三丈,踏着一柄青芒长剑,掠过塌方的矿坑落下。落地时靴尖不沾尘,袖口翻起,露出一线银线绣的纹样。

双蛇缠脉。

沈砚的呼吸慢了半拍。这个纹样他认得——清脉司缉拿队专用的记号,三年前他从一具挂在坊市示众的尸首背上见过一模一样的一枚。他们是来抓炉鼎的,而这矿场里,逆息体质的只有他一个。

他缩回井口后的暗影里,把矿牌塞进靴筒。晒谷坪上,为首那人正掀开一个矿工的下巴查看瞳色——清脉司验的就是逆息者吸灵时泛青的眼底。

一个一个验过来,迟早轮到他。

沈砚摸黑退向三号矿道。那条道通着废弃的排水暗渠,是他早就替自己留好的一条退路。他一边退一边记下:三人、御剑、验瞳。逃,得趁他们验到自己之前。
