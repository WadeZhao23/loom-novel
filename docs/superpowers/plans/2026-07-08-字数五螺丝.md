**Goal:** 用确定性反馈环治字数飘忽——注入实测字数给压缩授权装牙、超长改主动、细纲字数校验、陈旧细纲提醒、模板去硬编码、evals 严口径;做完发版 0.3.7。

**Tech Stack:** Python 3.11、pytest(conftest `project`/`FakeBackend`)、golden 回放测试。

**Spec:** `docs/design/proposals/字数五螺丝.md`。螺丝②(回炉不扩写)三期 T6 已做,本期不碰。

```python
_SCENE_BUDGET_RE = re.compile(r"约\s*(\d+)\s*字")


def _parse_scene_budgets(outline: str) -> list[int]:
    """抓细纲里各场「约X字」标注(螺丝④⑤共用)。没标返回空列表。"""
    return [int(m) for m in _SCENE_BUDGET_RE.findall(outline)]


def _check_scene_budget(project_root: Path, chapter_n: int, outline: str, chapter_target: int,
                        reused: bool, progress: Progress) -> None:
    """细纲字数确定性校验(非阻断,只 warn/info)。任何异常吞掉,绝不拖累出稿。
    reused=True(WYSIWYG 沿用已有细纲)→⑤陈旧提醒;False(大纲师刚产出)→④缺标注/偏差。"""
    try:
        budgets = _parse_scene_budgets(outline)
        total = sum(budgets)
        if reused:
            if total > 0 and chapter_target > 0 and abs(total - chapter_target) > chapter_target * 0.3:
                progress(events.info(
                    f"这份细纲按旧字数(约{total}字)拆的、当前目标 {chapter_target} 字,"
                    "建议在「本章细纲」点『重新生成细纲』按新目标重拆。"))
        else:
            if not budgets:
                progress(events.warn("细纲各场没标「约X字」篇幅预算,写手篇幅可能失控——可重新生成细纲。"))
            elif chapter_target > 0 and abs(total - chapter_target) > chapter_target * 0.3:
                progress(events.warn(
                    f"细纲各场字数总和(约{total}字)与本章目标({chapter_target}字)差得多,写手篇幅可能失控。"))
    except Exception:
        pass
```

run_pipeline 大纲师棒接线——在 `if outline_path and ...:` / `else:` 两分支各接一句(只对大纲师、`spec.outline_wysiwyg` 为真时)。WYSIWYG 分支(沿用已有细纲,`progress(events.info(...沿用...))` 之后):

```markdown
## 0.3.7 有什么

「设了 2000 字,写出来还是忽长忽短」——这一版给字数控制拧上几颗确定性的螺丝。

**字数更听话**
- 编辑/润色现在**知道原稿实际多少字**(以前靠模型自己数,数不准):超了目标就带着「超 X%」去压,压回目标量级、不动情节。
- 改了目标字数后,如果还在用**按旧字数拆的细纲**,会提示你「重新生成细纲」——这是改了字数却不听话最常见的原因。
- 细纲没给各场标字数、或总和跟目标差太多,会提醒你篇幅可能失控。
- 超长提醒更早(超目标 1.25 倍就提,不再等 1.5 倍)、更显眼(专门弹一下,不只默默记留痕)。

**说句实话**:AI 没法边写边精确数字数,单章还会有 ±25-30% 的波动——这是模型的天花板。这一版把「数字数」从模型手里拿回代码手里,尽量逼近你设的目标,但做不到分毫不差。想要某章篇幅精确,手改最稳。

Mac / Windows 都适用,无破坏性改动;老书零迁移。
```
