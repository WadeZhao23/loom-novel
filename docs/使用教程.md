# Loom 使用教程 · 从下载到写完第一章

> 一句话：Loom 是一台「织布机」——一队分工的 AI（设定师 → 大纲师 → 写手 → 编辑 → 润色师）读着你的**外置大脑**，一键织出一章正文；你手改，它**越写越像你**。稿子全在你自己电脑上。

这份教程给**不写代码的网文作者**看。跟着走一遍，二十分钟能写出第一章。

## 1. 先记住一张图：Loom 的核心循环

```
外置大脑（世界观·人物·卡章纲·你的文风）
        │
        ▼
   写下一章  →  你手改  →  「学这章的手改 learn」
        ▲                          │
        └──────  下一章更像你  ◀────┘
```

只要记住这条循环，其它都是细节：**AI 写 → 你改 → 它学你的改动 → 下一章更像你**。
它只学**你改的地方**，绝不学 AI 自己写的东西——所以越写越是你的味儿，不是越写越像 AI。

## 2. 第一步：下载并打开

去 [Releases 页面](https://github.com/WadeZhao23/loom-novel/releases/latest) 下载对应系统的包：

**Mac**
1. 下载 `Loom-mac.zip`，解压。
2. **右键点 Loom → 打开**（不是双击）。第一次会提示「无法验证开发者」，右键打开就能绕过——因为它没做付费签名，不是有问题。
3. 之后正常双击即可。

**Windows**
1. 下载 `Loom-win.zip`，解压，双击 `Loom.exe`。
2. 如果弹蓝色 SmartScreen 框：点「**更多信息 → 仍要运行**」。
3. 万一原生窗口起不来（个别精简版 Windows 缺组件），Loom 会**自动退回用浏览器打开**，功能一样，不影响使用。

> 打开后你会看到欢迎页。先别急着建书，可以点 **「先看看样例（一本跑通的书）」** 拷一本已经写好的示范书《重生记忆》进来，点开各文件看看「一本正常的书长什么样」，心里有底再自己建。

## 3. 答题起书，五分钟跑通第一章（最重要）

顺序说明：先在伙伴面板把「定位 / 世界观 / 人物 / 章纲」四项答齐 → 再写第一章（第 4 节细讲伙伴面板；这四项也可以像下面 ③ 那样让 AI 起草，两条路都算数，只要文件里有内容）。

### ① 新建一本书
欢迎页 →「新建一本书」：
- **放在哪个文件夹**：默认 `~/Desktop`，一般不用改。
- **书名**：随便填，例如「我的第一本书」。
- **题材（可选）**：选一个（都市/玄幻/言情…），Loom 只会把对应的一份「题材速查」拷给设定师参考；不选也完全能写。
- 点 **「新建项目」**。

### ② 配好后端（让 AI 能干活）
进主界面后看**顶栏**，选一个「后端」：
- **最简单、不用梯子**：选 `DeepSeek` → 把你的 API Key 粘进 **API Key** 框 → 点 **「保存后端」**。（怎么拿 key 见 [第 7 节](#7-选后端--填-keydeepseek-与免-key-的-claudecodex)，几块钱能写很久。）
- **已经在用 Claude Code / Codex 的**：后端切到 `Claude Code` 或 `Codex(GPT)`，**不用填 key**——它复用你电脑上客户端的登录（含订阅）。

填完点一下 **「检测连接」**（DeepSeek 是确认 key 在、Claude/Codex 是确认命令装好登录过），绿了就行。

### ③ 备好外置大脑（可以让 AI 起草）
左栏「**外置大脑**」里有：世界观、人物卡、卡章纲……点开**世界观**（或人物卡 / 卡章纲），如果是空的，会弹一个引导：
- 想省事：写**一句话设定**（例：「灵气复苏的都市，外卖小哥觉醒了能看见别人头顶任务的金手指」）→ 点 **「✨ 生成初稿（AI 起草，你再改）」**。AI 会照书名+题材起草 **世界观 / 人物卡 / 卡章纲** 三件套，**只填空白、绝不覆盖你已经写的**。
- 想自己写：点「我自己填」，直接在编辑器里写。写得越具体，AI 越不跑偏。

> 起草出来的初稿一定要**过一遍、改成你自己的**——这是你这本书的地基。

### ④ 喂一段你写的字（让它学你的文风）
左栏「外置大脑」下方点 **「喂样本，让它懂你（seed）」**：
- 选「**喂一段你写的字**」，贴 2–3 段你平时写的文字（越像你越好），点 **「提炼指纹」**。
- Loom 会从中提炼出你的「**写作指纹**」，之后写手就照这个味儿落字。

（没有现成的字也没关系，可以先跳过，直接写第一章，之后靠手改+learn 慢慢教它。）

### ⑤ 写第一章
左栏「正文」区点 **「写下一章」**。会弹出实时面板，你能看到 5 个 Agent 依次点亮：

> 设定师（立规矩）→ 大纲师（搭骨架）→ 写手（落字）→ 编辑（挑硬伤）→ 润色师（去AI味）

跑完点 **「完成，去看正文」**，正文就出现在编辑器里了（AI 还会自动起个章节标题放在第一行）。

### ⑥ 手改 + learn（关键一步）
- 在编辑器里**按你的喜好改这章**：改词、改句、加你的口头禅、删啰嗦……改完点 **「保存」**。
- 然后点 **「学这章的手改（learn）」**。会弹窗告诉你「这次学到了什么」：**金底=新增**（更像你）、**红底划线=删掉**。学歪了？点 **「撤销这次 learn」** 一键还原。

从第二章起，写手就带着你上一章的改动味儿写了。**这就是越写越像你的循环。**

## 4. 答题起书(伙伴面板)

新书打开后,左侧「伙伴 · 起书访谈」会一段一段问你:定位、世界观、人物、章纲——每题给几个选项,也可以自己写;你确认的答案直接写进外置大脑对应文件(和你手写的一样,随时可改)。想回头改点那一段的名字。起书四段不能跳过(它们是开写地基);voice 段可跳。四段齐了面板自动收起。

## 5. 界面速览：每个按钮干嘛

### 顶栏（后端与全局）
| 按钮 | 作用 |
|---|---|
| 后端下拉（DeepSeek / Claude Code / Codex(GPT) / OpenAI 兼容） | 选谁来写。随时切换 |
| 模型下拉 / 手动输入模型名 | 选具体型号，或自己填一个 |
| **拉取可用模型** | 拉这个供应商当前真实可用的型号（只读、不花钱） |
| **检测连接** | 确认后端就绪（不花钱） |
| **终稿字数** | 每章目标字数 |
| **保存后端** | 把后端设置存进这本书 |
| 明暗主题 / 专注模式 ⌘. | 换皮肤 / 只留编辑器沉浸写 |
| **自检** | 一键检查 key、命令、大脑文件是否齐（缺什么会告诉你怎么补） |
| **回首页** | 退出这本书，去建/开别的书 |

### 左栏（这本书的一切）
- **正文**：章节列表；**写下一章**；**导出全书**（一个 txt）；**备份整本**（打包 zip）。
- **外置大脑**：世界观 / 人物卡 / 卡章纲 / 立项卡 / 文风参考 / 违禁词，加上你的 **写作指纹** 卡；**喂样本，让它懂你（seed）**。
- **skills（方法论）**：跨书通用的写作套路（网文大神 / 去AI味 / 故事引擎 / 黄金开篇…），一般不用动。
- **agents（5 道工序的提示词）**：点任一个能看「一章是怎么织出来的」，好奇再看。

### 编辑器上方（针对当前这一章）
| 按钮 | 作用 |
|---|---|
| **预览** | 渲染好看的排版 / 纯文本编辑，来回切 |
| **历史** | 本章版本历史，随时**回滚**（每次覆盖前自动存，最多 30 份） |
| **违禁词** | 本地粗筛平台敏感词，只提示不阻断 |
| **本章细纲** | 看/改本章分镜骨架 |
| **重新生成细纲** | 换个骨架方案 |
| **重写选中** | 选中一段，按你的嗓音重写（只改这段，剧情默认不动） |
| **学这章的手改（learn）** | 把你的改动蒸馏进写作指纹 |
| **保存** | 存当前文件 |

> 小技巧：按 **⌘K**（Win 是 Ctrl+K）打开命令面板，直接搜「写下一章 / 重写 / 导出 / 主题…」。章内查找按 **⌘F**。

## 6. 外置大脑：这本书的记忆

「外置大脑」是**每本书独有、会随剧情长大**的记忆。5 个 Agent 写作时都读它。共 6 个文件：

| 文件 | 装什么 | 谁维护 |
|---|---|---|
| **世界观.md** | 设定、境界/等级阶梯、金手指规则、地名势力 | 你 + 可 AI 起草 |
| **人物卡.md** | 人物档案、关系、专名 | 你 + 可 AI 起草 |
| **卡章纲.md** | 分卷分章大纲、爽点、伏笔 | 你 + 可 AI 起草（learn 后还会自动回填每章摘要） |
| **立项卡.md** | 平台/分区/题材/对标意图（可空，见第 10 节） | 只你手填 |
| **文风参考.md** | 别人的范文样例，当风格佐料（见第 10 节） | 只你手填 |
| **违禁词.md** | 你这本书要回避的词 | 你 |
| **写作指纹.md** | 你的文风（seed / learn 自动养成，别手写覆盖） | Loom 养成，可撤销 |

要点：
- **硬设定原样传**：境界名、金手指代价、地名、人物专名，Loom 会把原文**整段直送**给大纲师和写手，一字不改——这样第 2、3 章不会把「一中」写成「二中」、凭空多出一个等级。
- **只追加、不覆盖**：AI 补充的内容会以「AI 补充」区追加在文件末尾，你手写的部分永远不动。

## 7. 选后端 / 填 key：DeepSeek 与免 key 的 Claude、Codex

Loom 自己**不卖 key、不做中转**。用哪个后端你自己定：

### A. DeepSeek（推荐给大多数人：国产、不用梯子、自带 key）
拿 key 四步：
1. 浏览器打开 **platform.deepseek.com**，注册 + 实名。
2. 充值（按字数计费，几块钱能写很久）。
3. 在「API Keys」里创建一个 key（`sk-` 开头），复制。
4. 回 Loom 顶栏，把 key 粘进 **API Key** 框 → 点 **「保存后端」**。

模型：默认 **V4 Pro · 更强**（最稳）；想更快更省选 **V4 Flash**。模型名不用背——列表里没有就点「拉取可用模型」拉真实的。

### B. Claude Code / Codex（免 key）
如果你电脑上已经装了 `claude` 或 `codex` 命令并登录过：
- 后端切到 `Claude Code` 或 `Codex(GPT)`，**不用填任何 key**，Loom 直接复用它们客户端的登录（含订阅）。
- Codex 的模型建议留在「默认（codex 自己选）」，不用手动指定。

### C. OpenAI 兼容（自定义）
接智谱 GLM / Moonshot(Kimi) / 通义 Qwen / 硅基流动等任意 OpenAI 兼容供应商：选 `OpenAI 兼容(自定义)` → 填 **base_url** + 模型名 + 该供应商的 key → 保存。

> 换后端不会互相清掉 key，各存各的，随时切回来。

## 8. 「越写越像你」是怎么发生的

这是 Loom 的灵魂，值得单独讲清楚：

- **写作指纹 = 你的嗓音**。写手和润色师照它落字。
- 指纹**只从你的手改里学**——你点 learn，它把「你改了什么」蒸馏进指纹。**它绝不学 AI 自己写的输出。**
- 指纹是**累积**的：只增不删，换模型也不会把你攒下的味儿冲掉；万一某次 learn 学歪或磨短了，弹窗里 **「撤销这次 learn」** 一键还原。

**正确姿势**：每写完一章，认真手改（哪怕只改几处关键句），再 learn。改得越像你，教得越快。别嫌麻烦——前几章多教，后面越省心。

## 9. 质量护栏：违禁词 / 去AI味 / 质检 / 伏笔

Loom 的护栏都是**只提示、不替你改、不阻断出稿**（默认），你永远说了算。（写作中的护栏不阻断；唯一例外是**开写前的起书地基**——立项/世界观/主角/章纲没齐，不给写第一章。）下面这几道，分别管什么：

- **去AI味**：润色师这一棒专门擦「机器腔」，让文字像真人。它是独立功能，只擦通用 AI 味，**不针对任何检测器作弊**。你签过字（learn 进指纹）的句子会被豁免，不会被当 AI 味擦掉。
- **质检**：编辑挑硬伤（爽点/钩子/人物 OOC/信息穿帮——比如角色「知道」他当时不在场的事）。默认跑 **1 轮，只诊断、不回炉**——把问题留痕给你看，改不改你定。想让它自动回炉重改，可以把轮数调到 ≥2。
- **违禁词自检**：编辑器上方点「违禁词」，本地粗筛平台敏感词 + 给改写建议，只提示。
- **伏笔悬空提醒**：埋了很久却没回收的伏笔，会在《审稿留痕》里提醒你（默认隔 8 章还没还才提）。
- **审稿留痕**：编辑改动的记录另存在项目里的 `.审稿留痕/`，**绝不进终稿、不污染你的指纹**，纯给你回头看。

## 10. 进阶玩法：立项卡 · 文风参考 · 播种别人的范文

这三条不是一回事：**立项卡跟世界观、人物、卡章纲一样属于「答题起书」的地基**——开写第一章前得有内容（各格本身仍可空，填了一处就算完成；见 [第 4 节](#4-答题起书伙伴面板)）。**文风参考、播种别人的范文才是真正随时可选**，留空完全不影响出稿：

- **立项卡**（外置大脑里的 `立项卡.md`）：填平台 / 分区 / 题材 / 对标意图 / 为什么选它（各格可空）。整张卡喂设定师当「定位背景」，平台字段还会给违禁词自检定松紧。**只你手填，Loom 从不自动写。**
- **文风参考**（`文风参考.md`）：贴**别人的**原文范文当风格样例，每章进写手当佐料。三重兜底护着你：指纹永远压范文、只学节奏措辞句式、**严禁照搬专名整句**，而且**范文绝不进 learn**——所以「开局像他、终局仍收敛到你」。
- **播种别人的范文（seed --参考）**：可以从你欣赏的作者原文**蒸出一个「起点」指纹**，此后靠你自己的改稿慢慢脱化。种子只定起点，真正的学习信号**永远只有你的手改**。

> **更新的功能**（本教程尚未逐一展开，详见 [README 更新记录](../README.md#更新记录)）：正文**行内改**（鼠标划到某行浮出「改」）、**跨章除虫**（每章写完自动查物品复活/人设跌份/时间线等矛盾 + 状态账本）、**导入铺底**（选个文件夹把已有设定/大纲接成新书）、**字数更听话**（编辑知道原稿实测字数才压、改字数后旧细纲提醒重生成）、**品牌楷体随包**。

## 11. 日常操作：改章、回滚、局部重写、导出、备份

- **写第 N 章**：写第 N 章时，AI 会自动读你手改后的第 N−1 章做衔接，所以先把上一章改好再往下写。
- **章节管理**：左栏每章可删 / 插空章 / 上下移，误删进回收站可恢复。
- **回滚**：编辑器上方「历史」→ 挑一个旧版本预览 / 回滚（回滚前会把当前版也存一份，能再回来）。
- **局部重写**：选中一段 → 「重写选中」→ 可给一句指令（如「太铺张压一压」「换个更冷的章末钩」）→ 满意点「用这个」。AI 重写会进快照、**不污染 learn**。
- **导出全书**：左栏「导出全书」出一个 txt，直接拿去发平台。
- **备份整本**：左栏「备份整本」打包成 zip。**这个 zip 不含你的 key**，拷给别人 / 传网盘是安全的。

## 12. 稿子安全 & 隐私

- **没有服务器、没有账号、没有遥测**：Loom 自己不收集、不上传任何东西。
- **你的书全在本地**：正文 / 外置大脑 / 写作指纹 / 版本历史 / 备份，都在你磁盘上的项目文件夹里。
- **AI 生成时会把上下文发给你选的模型**（任何 AI 写作工具都绕不开）：写一章时，世界观/人物/卡章纲/指纹/上一章+你的指令会发给 DeepSeek 云端（用你自己的 key）或经你本机 Claude/Codex 客户端。Loom 只是转交，不额外留存。
- **关于 key**：DeepSeek 的 key 明文存在项目里的 `.env` 文件。**别把含 `.env` 的整个项目文件夹发给别人**；用「备份整本」生成的 zip 不含 `.env`，拷走是安全的。

## 13. 常见问题 & 报错怎么办

**Q：一定要会写代码 / 装 Python 吗？**
不用。普通作者直接下 Releases 里的 Mac / Win 包，双击（Mac 右键打开）就能用。

**Q：DeepSeek 提示「返回空内容 / 这一步没产出」？**
DeepSeek V4 是「先思考再写」的模型，偶发思考占满额度导致正文空。新版已给思考留足额度、默认也用更稳的 V4 Pro。**遇到就重写那一步即可**——你的指纹和已写章节始终【原样保住】，不会被空内容覆盖。

**Q：Claude / Codex 检测连接不过？**
确认你电脑上确实装了 `claude` / `codex` 命令并**登录过**（Codex 要 `codex login`）。点顶栏「自检」会告诉你缺什么、怎么补。

**Q：模型名填错了会怎样？**
只会软提示、不阻断。拿不准就点「拉取可用模型」拉真实列表，别凭记忆填。

**Q：learn 之后感觉文风变怪了？**
在 learn 弹窗点「撤销这次 learn」一键还原。指纹是累积的，撤销不影响之前攒的。

**Q：Windows 双击闪退 / 弹一串报错？**
新版会自动退回用浏览器打开，功能一样。若仍不行，先更新到最新版再试。

**Q：写一章要多久？**
5 道工序跑完通常一到几分钟（跟字数、后端有关）。用 Claude/Codex 时长章可能更久，耐心等实时面板跑完即可。

**Q：怎么让它更像我？**
多写、每章都认真手改再 learn。前 3–5 章是「教它」的关键期，之后就顺了。
