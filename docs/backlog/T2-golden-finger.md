## 可落盘内容(蒸馏成品)

# 金手指设计(给设定师)

> 你的活:帮人把"主角凭什么逆袭"这件事,从一句模糊的设想,落成 外置大脑/世界观.md 里**可遵守、有代价、能持续供爽**的几行字段。
> 这是"写什么(what)"层的手艺——只管金手指的规则与剧情功能,**不碰任何文风/句法/语气**(那是写作指纹的事,绝不在这里出现)。
> 金手指不是越强越好,是越"巧"越好:最好的金手指让主角"努力 + 外挂"双管齐下,既不显得不劳而获,又能爽快碾压。

## 三条底线原则
1. **有功能**:必须有一句话能说清的核心定位——它补主角哪块短板(修炼慢 / 资源缺 / 信息差 / 战力不足)。说不清就别要。
2. **有限制**:必须有硬代价或硬约束(冷却 / 消耗 / 触发条件 / 反噬)。**无限制 = 无冲突 = 没法写**。
3. **会成长**:随主角境界分段升级,避免一成不变失去新鲜感;但每次升级前后**必须兑现一次代价**。

## 必填七字段(逐条填进 世界观.md 的"金手指"段)
| 字段 | 它回答什么 | 怎么填(举例) |
|------|-----------|--------------|
| 核心功能 | 补哪块短板 / 解决什么 | 资源获取 / 信息差 / 战斗补足 / 升级加速 |
| 触发条件 | 何时生效 | 特定场景 / 冷却完成 / 支付代价后 / 情绪爆发 |
| 代价·限制 | 用它必须付出什么 | 折寿 / 失忆 / 耗灵石 / 透支信任 / 越级反噬(见代价库) |
| 反制方式 | 敌人怎么克制它 | 规则屏蔽 / 断供 / 同类镜像 / 误导利用(见反制库) |
| 可见度 | 读者/角色谁知道 | 明牌(都知) / 半明牌(读者知角色不知) / 暗牌(主角藏着) |
| 升级路径 | 能力怎么长大 | 三段式 or 版本迭代,绑主角境界:练气→筑基→金丹… |
| 爽点节点 | 哪几个高潮靠它 | 获得(仪式感) / 首次使用(立竿见影) / 升级解锁 / 关键时刻救命逆袭 |

## 代价库(至少选一种"硬代价",别只挑软的)
- 寿命/健康:每次使用折寿或伤身
- 记忆/情感:重要记忆丢失、情感冷却
- 资源消耗:灵石/积分/信用点等硬通货
- 关系代价:信任透支、人际崩塌
- 规则反噬:越级必反噬、强制冷却
- 信息代价:得一份情报必失等量情报

## 反制库(敌人不能对金手指束手无策)
- 规则屏蔽:敌对规则让金手指失效
- 断供:资源/系统点数被切断
- 镜像对抗:反派拥有同类能力
- 误导利用:系统提示被反向利用
- 代价转移:反派逼主角承担更多代价

## 供爽节奏(填升级路径/爽点节点时对照)
- 小回报:每 2-3 章至少 1 次可感知反馈
- 中回报:每 8-10 章一次明显升级
- 大回报:每 20-30 章一次版本/形态更迭
- 代价兑现:每次大回报前后必须兑现一次代价(否则爽感贬值)

## 七类型速查(拣一个起步,别从零硬想)
| 类型 | 核心爽点 | 典型代价 | 常见反制 |
|------|---------|---------|---------|
| 系统面板 | 数值成长/可视化 | 任务失败惩罚 | 规则屏蔽/同类系统 |
| 随身空间 | 囤货/资源跃迁 | 资源消耗 | 断供/空间封印 |
| 重生穿越 | 信息差/先知 | 记忆偏差/历史变动 | 历史改道/对手也重生 |
| 签到打卡 | 持续奖励 | 冷却/次数限制 | 限定场景/封印 |
| 器灵导师 | 指点/传承 | 恢复需资源 | 器灵沉睡/封印 |
| 血脉觉醒 | 强化跃迁 | 反噬/失控/静养 | 克制血脉/封印 |
| 异能觉醒 | 单点爆发 | 精神/体力反噬 | 反制异能 |
| 无金手指 | 转移到人设/选择/逆袭 | —— | 言情/现实向/短篇适用 |

> 可视化建议:系统/签到类适合用"面板/属性/进度条"给读者直观反馈(+10 力量 / 境界突破提示),但**面板是调味不是正文**,别整段堆数据。

## 获得方式(必须有理由,忌凭空冒出)
- 意外融合(捡到古物 / 被雷劈 / 濒死激活)
- 传承激活(血脉觉醒 / 祖传宝物)
- 系统降临(神秘声音 / 光幕弹出)

## 复合金手指(想叠两个时)
- 一主一辅,别"外挂叠外挂"。
- 代价不可同时归零:至少保留一种硬代价。
- 例:系统(管任务/升级)+ 空间(管资源)。

## 题材→金手指匹配(优先推荐,非强制)
| 题材 | 推荐金手指 |
|------|-----------|
| 玄幻修仙 | 系统面板、签到打卡、器灵传承、血脉觉醒 |
| 都市异能 | 系统面板、重生记忆、异能觉醒、随身空间 |
| 言情 | 重生记忆、随身空间、无金手指 |
| 短篇/知乎 | 单一特殊能力、无金手指 |
| 规则怪谈 | 系统提示、规则解读能力 |
| 末世 | 随身空间、资源兑换系统 |
| 无限流 | 任务系统、关卡规则提示 |

```
## 金手指
- 类型:重生记忆(暗牌)
- 核心功能:信息差——提前知道三年内的秘境/机缘/叛徒
- 触发条件:被动,记忆自带;关键情报靠"回想"消耗精神力
- 代价·限制:历史一旦被改动,后续记忆即失效;过度回想会头痛昏厥
- 反制方式:反派同为重生者,记忆相互失真;蝴蝶效应让既定事件改道
- 可见度:暗牌,主角刻意隐藏重生身份
- 升级路径:记忆碎片→完整时间线→看清幕后(绑主角境界三段解锁)
- 爽点节点:首次靠先知避劫(第3章)/ 抢先得机缘打脸天才(第12章)/ 发现对手也重生反转(第30章)
```
