# Loom 上手 · 一页版

**Loom 是什么**：一队 AI（设定师→大纲师→写手→编辑→润色师）读你的设定，一键织出一章正文；你手改，它**越写越像你**。稿子全在你自己电脑上。

---

### 六步跑通第一章

1. **下载打开** — 去 [Releases](https://github.com/WadeZhao23/loom-novel/releases/latest) 下 Mac / Win 包。Mac **右键→打开**；Win 双击（蓝框点「更多信息→仍要运行」）。
2. **新建一本书** — 欢迎页填书名 → 「新建项目」。（想先看范例：点「先看看样例」）
3. **配后端**（二选一）
   - **DeepSeek**：platform.deepseek.com 注册充值拿 key（`sk-` 开头，几块钱写很久）→ 粘进顶栏 **API Key** → 「保存后端」。
   - **Claude / Codex**：顶栏后端切过去，**不用填 key**（复用你电脑客户端的登录）。
4. **备好外置大脑** — 左栏点「世界观 / 人物卡 / 卡章纲」，空的会弹引导：写**一句话设定** → 点 **「✨ 生成初稿」** 让 AI 起草，再改成你的；或「我自己填」。
5. **喂样本** —（可选）左栏 **「喂样本，让它懂你（seed）」** → 贴 2–3 段你写的字 → 「提炼指纹」。它就懂你的味儿了。
6. **写 → 改 → 学** — 左栏 **「写下一章」**，看 5 个 Agent 跑完 → 在编辑器**手改**、点「保存」→ 点 **「学这章的手改（learn）」**。

> **越写越像你**：它**只学你手改的地方**，绝不学 AI 自己写的。前几章多改多 learn，后面越省心。

---

### 三个必须知道

- **改坏了能回退**：learn 学歪 → 弹窗点「撤销这次 learn」；正文改坏 → 「历史」里回滚。数据不会丢。
- **DeepSeek 偶尔「返回空内容」**：重写那一步即可，你的指纹和已写章节始终原样保住。
- **稿子在本地、key 别外传**：整个项目文件夹含你的 key，别直接发人;要分享用「备份整本」的 zip（不含 key）。

---

**转发一句话**：下 Releases 的 Mac/Win 包 → 新建书 → 填 DeepSeek key（或切 Claude/Codex 免 key）→ AI 起草设定、喂段你写的字 → 「写下一章 → 手改 → learn」。改得越多，越写越像你。
