# 领航员头像:水墨卡通「提灯小童」——生成留档

2026-07-13 · 作者两轮拍板:第一批与五工序同风格的写实白描(5 张)被否(「弄个水墨风卡通形象吧」),第二批 Q 版水墨卡通中选 **chibi_a / seed 42**(侧身提大纸灯,暖光,发髻,腮红)。

## 最终提示词(ModelScope Qwen/Qwen-Image,1024×1024,seed=42)

> 中国水墨风格的Q版卡通形象,可爱的大头小身比例,圆圆的脸蛋,温和的笑眼,宣纸留白背景,墨黑小长衫用湿笔淡墨渲染,头发是几笔简练的墨色,双手提着一盏比例偏大的圆纸灯笼在身前,灯焰一点朱砂红,画面中唯一的彩色就是这一点朱红,其余全部是墨色与宣纸白,极简可爱,大量留白,角色居中

要点(复现链,补五工序头像当年断掉的留档):
- **全正向表述锁色**——Qwen-Image 负向提示反成吸引子(仓内记忆实测教训);
- 生成走 **ModelScope 直连 API**(异步 `/v1/images/generations` 提交 + `/v1/tasks/{id}` 轮询 + urllib 自取 URL)——MCP `generate_image` 的下载步有 content-type 校验 bug,绕过;token 在 `~/.claude.json` mcpServers env `MODELSCOPE_SDK_TOKEN`;
- 候选批:3 措辞变体(提灯/举灯/抱灯)× 2 seed(42/7);挑选三关:风格贴合、红色克制、圆裁 24/40/64px 小尺寸可辨。

## 与五工序头像的风格关系

五工序是写实向白描淡彩(胸像、职业道具、一点朱砂红);领航员刻意走 **Q 版卡通**(作者拍板)——它是唯一直接跟作者对话的拟人角色,baby schema(大头圆脸)符合"陪跑伙伴"人设(调研:Duolingo Duo 公式);五工序是"幕后工序卡片",不与它同场竞争。红色纪律仍守:主红=灯焰(腮红为淡粉,属肤色渲染)。
