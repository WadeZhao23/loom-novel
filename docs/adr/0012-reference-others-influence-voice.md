# 参考别人的作品影响文风:L1 播种起点 + L2 文风参考入写手,但刻意不走 L3

> 状态:**已采纳并实现于 3.0.0**(文风参考模板 + 写手 reads + 阀②/护栏 prompt 指令;`loom seed --参考` 从别人范文蒸起点指纹)。

起因是重新起书时作者明确要求:"必须能参考别人的作品、结合起来产生好作品,'只能有你自己的指纹'不能是红线。" 而 Loom 原本对"参考别人"是**分层的**:what 侧早已全开(`拆书` 抽可迁移框架、`题材速查` 给套路),voice 侧则一条硬线——`写作指纹` 只从你的改稿学、`写手` 只读 `写作指纹`+`网文大神`,刻意不让别人的文风渗进落字(守 [ADR 0002](0002-fingerprint-purpose-and-division.md) 的"越写越像**你**")。作者要放宽的正是 voice 侧这条线。

## 决定:voice 侧开两档受控通道(L1+L2),第三档(L3)刻意不开

把"参考对文风的影响"排成一根光谱,取中间两档:

- **L1 · 播种起点**:`loom seed` 的来源扩到"**你欣赏的作者的原文范文**"——用它蒸出**起点**指纹(开局像他),此后靠你的改稿脱化。
- **L2 · 文风参考入写手**:新增可选外置大脑文件 `文风参考`,持有原文范文片段,每章进 `写手` 的 `reads:` 当风格样例。
- L1 与 L2 **可共用同一份素材**(你贴的作者原文):`seed` 拿它定起点、`文风参考` 留原文每章喂写手。

## 两个安全阀 + 一条护栏(它们是 L1+L2 不塌成 L3 的全部依据)

- **阀① 学习信号不变**:即便从别人样本起手、每章掺范文,`写作指纹` **仍然只从你的改稿学**(学习增量)。种子只定起点、范文只当样例,**都不是学习源**。这条保证"**开局像他、终局像你**"——指纹持续朝你收敛。**ADR 0002 不动。**
- **阀② 指纹压范文**:`文风参考` 与 `写作指纹` 冲突时,**指纹赢**(你是主、范文是佐料)。
- **护栏 · 专名/原句**:`文风参考` 只学节奏/措辞/句式,**不许搬专名或整句原文**(镜像 `拆书` 的专名黑名单),防"佐料"变"洗稿"。

## Considered Options(选摘)

- **L0 维持原线(voice 只准从你来)**——否决:作者判定太刚;真实作者练文笔本就"先模仿再脱化",一刀禁掉别人是人为洁癖。
- **L3 让写作指纹持续从别人的稿子学**——**刻意不开**:它会 supersede ADR 0002、把"学习信号唯一是你的改稿"推翻,指纹永久收敛到"你+别人",而"Loom 写出来像**你**"是这产品**唯一的护城河**。作者本人选 L1+L2 而非 L3,正是要留住这条。
- **复用 `拆书` 当范文源**——否决:拆书**刻意剥 voice**(抽 what、强制专名黑名单),与 L2"要 prose 喂 voice"正好相反;必须另立 `文风参考`。

## 与既有不变量的兼容

- **`拆书`(what 侧)与 `文风参考`(voice 侧)分工对立、永不合流**;两条都不回流写作指纹。
- **`题材套路→写手` 仍禁**:那是无意的 what 污染、无三重约束;`文风参考→写手` 是有意、受阀②+护栏+阀①约束的例外(见 CONTEXT 的「Flagged ambiguities」)。
- **去AI味/指纹护栏/关卡照旧**:范文不改 learn、不打分、不阻断;`文风参考` 同 `违禁词.md`/`立项卡`——外置大脑里人手维护、loom 从不自动写。
