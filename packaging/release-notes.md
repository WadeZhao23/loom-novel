把一队分工 Agent 织成一条**写小说的流水线**,做成桌面客户端。读你的"外置大脑"一键跑出一章正文;你手改,它**越写越像你**。你的书和写作指纹都存在本地,Loom 自身不收集、不上传;只有用 AI 写作时,才把必要上下文发给你选的模型(DeepSeek 云端 / 本机 claude·codex / 任意 OpenAI 兼容供应商)。自带 DeepSeek key,按字数计费。

## 1.3 修了 / 新增

- **再也不会「换模型把写作指纹学空」**(重要修复):有人换了个模型名后 `learn`,结果把攒下的写作指纹覆盖成空。根因是模型返回空内容时被原样写盘。现在所有「模型输出 → 覆盖你的文件」都先校验:**这次没产出有效内容,就保留原文件、明确报错,绝不拿空的覆盖你**;指纹疑似被磨短还会提示「可一键撤销」。
- **模型改成下拉选择,不再手填**:顶栏模型从输入框换成**下拉点选**,只给该供应商正确的型号——不会再因为模型名填错(比如把 `v4-flash` 当 DeepSeek 名)而出问题。DeepSeek 升级到 `deepseek-v4-flash` / `deepseek-v4-pro`(旧名 `deepseek-chat/reasoner` 官方 2026-07-24 停用,已从列表移除)。
- **多供应商路由**:新增「OpenAI 兼容(自定义)」后端,自填 base_url + key 即可接智谱GLM / Moonshot Kimi / 通义Qwen / 硅基流动等;点「拉取可用模型」实时列出这家真实可用的型号。
- **每章带标题**:写完一章自动起个标题、落进正文首行,侧栏直接显示,随时手改;改标题不会被当文风学进指纹。

## 1.0.1 修了什么

- **修 Windows 打不开**:部分 Windows 上双击 `Loom.exe` 直接弹错(`Unable to configure formatter 'default'`)闪退。原因是无终端窗口的构建里程序没有标准输出,内部日志初始化时崩掉。已修。Mac 版不受影响,无需更新。

## 1.0 这版有什么

- **不丢稿**:所有写盘原子化(断电/崩溃不会把正文截成空);每章自动留版本历史,误删误覆盖一键回滚。
- **章节管理**:侧栏每章可删 / 插空章 / 上下移,关联产物自动重排,删除进回收站可恢复。
- **三种后端**:DeepSeek(自带 key)/ Claude Code / Codex(复用客户端登录免 key),一键检测连接。
- **角色化的 5 道工序**:设定师→大纲师→写手→编辑→润色师,每个 Agent 有形象人设,点开看整条流水线。
- **违禁词自检**:按国内平台常见雷区本地粗筛,只提示不阻断;改写指引随书附带。
- 全套图标重做、明暗主题、新手引导。

## 怎么用(不写代码也能用)

**Mac** —— 下载 `Loom-mac.zip` → 解压得到 `Loom.app` → **右键 → 打开 → 再点"打开"**(本版未签名,首次必须右键打开,绕过"无法验证开发者";直接双击会被拦)。

**Windows** —— 下载 `Loom-win.zip` → 解压 → 进 `Loom` 文件夹双击 `Loom.exe`(SmartScreen 蓝框时点"更多信息 → 仍要运行")。

打开后:**先点「📖 看看样例」**看一本跑通的书;要写自己的,按欢迎页「🔑 怎么拿 DeepSeek key」填上 key(自带 key,平台按字数计费,几块钱写很久)。
