🔑 怎么拿 DeepSeek API key(自带 key)

Loom 用你自己的 key 写作,稿子不离开你的电脑。四步:

  1. 浏览器打开 platform.deepseek.com,注册 + 实名
  2. 充值(按字数计费,几块钱能写很久)
  3. 在「API Keys」里创建一个 key(sk- 开头)
  4. 新建/打开书后,把 key 粘进顶栏 API Key 框 → 点「保存后端」
