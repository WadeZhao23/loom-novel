# 世界观

## 一句话定位
灵气将枯的末法修仙界,大宗门攥着最后几条灵脉;一个被测出废灵根的杂役,藏着重活一世的记忆。

## 力量体系
- 炼气九重楼:练气 → 筑基 → 金丹 → 元婴。如今金丹已是一宗之祖。
- 灵气稀缺到要配给,杀人夺宝是常态——怀璧其罪,出门先掂量"谁能杀我"。

## 金手指
- 类型:重生记忆(暗牌)
- 核心功能:信息差——他记得未来三年内每一处机缘、每一场死、每一个会倒戈的人。
- 触发条件:被动;细节靠"回想",回想多了头痛欲裂。
- 代价·限制:历史一旦被他改动,那之后的记忆就作废;过度回想会当场昏厥。
- 反制方式:不止他一个人重生,两份记忆会相互失真;他一动手,蝴蝶效应就让既定的事改道。
- 可见度:暗牌。他宁可挨打,也绝不让任何人看出他"知道"。
- 升级路径:零碎记忆 → 完整时间线 → 看清抽干灵气的幕后黑手。
- 爽点节点:首次靠先知避杀劫 / 抢在天才前头摘机缘 / 撞见对手也是重生者。

## 地理 / 势力
- 青石宗:没落二流宗门,守着一条快干的灵脉,内里勾心斗角。
- 后山乱葬岗:埋死了的杂役,没人去。

## 冰山真相
- 灵气枯竭不是天灾。是有人在抽。
