# 写作指纹

> 这是"你"的文风规则化描述；写手/润色师写前会读它，你 learn 后它会更新。目的只有一个：越写越像你。

## 句式偏好
- 极短句主导，单句成段是常态；三字以下的句子独立成行，制造锤击感
- 动作串联不加连词，直接逗号堆叠动词：「舔掉——坐下——闭眼」，没有"然后"
- **首动作可单独成句**：动作序列里第一个动作有时拆出来独立成句，随后的动作才串联——「他起身。叠被，理薄毯。」
- 情绪不解释，用行为替代：不写"他很平静"，写"声音很平"
- 单字/短语成段，像镜头切割：「一下。两下。」
- 结尾句留白极重，最后一句往往是悬崖边的单句
- **并列否定句各自成句**：「不是忍。是不在意。不是不疼。疼也不值一个表情。」——逗号并排是AI写法，句号切断是作者写法

## 口头禅 / 高频表达
- 「他」字反复作主语，不换代称，冷距感
- 「很 + 单字形容词」作唯一修饰（很平、很稳）——极简评价
- 数字递进（一下、两下、三年）作节拍器和具体感来源
- 否定句做转折揭示：「那不是……」
- **「没人」优于「没有人」**——负性存在句一律压缩（「没声音」不写「没有声音」）

## 禁用词 / 绝不这么写
- 不用「突然」「忽然」「猛地」等副词起势
- 不用感叹号，不用省略号（靠句号制造停顿）
- 不写心理独白（「他心想……」「他不禁想到……」）——内心只从行动泄露
- 不用叠词渲染（慢慢地、静静地、陆陆续续）
- 不用长段铺垫或过渡句，不解释因果
- **不用「就这么」作过渡填充**：「他就这么跪着」→「他跪着」
- **不用进行态"在"**：「不是在忍」→「不是忍」
- **不用「真的」「的确」作强调**——断言直接落
- **不加「地」作副词后缀**：「一鞭一鞭打下去」不写「一鞭一鞭地打下去」
- **不保留可删去的修饰语**：多余的环境定语直接删，裁到最小可辨单位

## 节奏
- 整体：**短-短-短-稍长-短** 的锯齿结构，偶尔一句稍长句做缓冲，随即被短句收死
- 停顿靠句号，不靠标点花样；密集句号营造喘不过气的压抑感
- 结尾必然减速：最后一段节奏放慢，留一句话的空白让读者自己填
- 克制优先于渲染——写七分，留三分不说
