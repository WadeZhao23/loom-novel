# 科幻题材模板

> 给【设定师】读,用来圈这本书的剧情边界。管「写什么套路」(what),不碰「怎么写得像你」(voice)。
> 参考速查,不是铁律——和你的脑洞冲突时以你为准,直接手改。

## 核心爽感
在硬核科技与星辰大海里,用超前技术和宏大想象碾碎旧秩序,看主角把人类文明的边界一寸寸往外推。

## 读者期待(不交付会弃书)
- 设定有"未来感"且自洽:科技等级、社会形态、势力格局摆得清楚,不是披科技皮的玄幻。
- 选定一条细分调性并坚持(赛博朋克的高科技低生活 / 太空歌剧的星际征伐 / 硬科幻的逻辑推演 / 末日求生 / 机甲热血)。
- 有真实的伦理与冲突重量:克隆、意识上传、AI觉醒、人机矛盾这些命题被认真对待,不是背景板。

## 必备桥段(选用,别全塞)
- 技术碾压:主角握有超前技术(穿越/传承/天才),用高科技吊打低维敌人。
- 机甲/星舰觉醒:与机甲人机合一战力飙升,或主角成舰长战术指挥以少胜多。
- 义体升级 / 黑客入侵:在赛博都市里改造肉身、侵入系统、反抗巨型企业体制。
- 第一次文明接触:遭遇外星文明,文化冲击、技术交流、危机化解。
- 绝境救世:科技灾难或终极威胁压顶,靠科学手段扭转战局、拯救文明。

## 常见雷区(踩了掉粉)
- 科技设定前后矛盾:开篇定好硬规则(能源、推进、武器上限),后文别为爽点随手破例。
- 术语堆砌读者看不懂:新概念随剧情逐步解释,别一上来甩一墙黑话。
- 科技万能没有代价:每项强科技配上代价(能源消耗、义体排斥、基因崩溃、AI失控)。
- 外星人太像地球人:异文明要有真异质感(硅基、集群意识、能量生命),不是换皮人类。

## 金手指匹配
- 形态:超前科技/外挂系统/特殊机甲或星舰/被改造的身体——本质是"信息或技术代差"。
- 成长性:从单星球摸索到星系、银河乃至宇宙级,金手指随科技树解锁逐级升维。
- 必须有的限制/代价(防无敌):来源要交代清楚(穿越/传承/天才);用强科技必付代价——能源枯竭、义体排斥与失人性、基因/意识崩溃、AI叛变或武器失控。指挥型爽点要靠展示战术才能,不是无脑碾压。
