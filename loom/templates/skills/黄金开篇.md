# 黄金开篇(给写手 · 只在写第 1 章时加载)

> 第一章前 500 字决定读者去留。第一句话的唯一任务:让读者读第二句。

## 四大开篇模型(选一种或组合)
1. **死亡倒计时**:第一段就把主角扛到死局,没有铺垫;反常地活下来,带出新悬念。
2. **极致背叛**:先展示深度信任 → 粉碎式背叛 → 主角被踩到尘埃 → "眼神变了"。
3. **信息炸弹**:开篇抛一个"不该存在"的事实,不解释,制造必须填的信息缺口。
4. **身份拉扯**:主角藏着秘密身份,被轻视的场景里不经意露一手,读者获得"我知道他是谁"的优势。

## 第一句话公式(任选)
反常识句 / 倒计时句 / 感官爆炸句 / 冲突对话句 / 结果前置句。

## 绝对禁止的开篇
- "我叫 XX,今年 XX 岁" 式自我介绍
- 大段世界观说明(融进行动和困境里展示)
- 天气/环境抒情铺垫
- 主角的日常流水账(从日常被打破那一秒开始写)

## 黄金三章要在前三章内完成
建立代入感 + 埋下贯穿全文的超级钩子 + 释放第一个小爽点 + 章末弹出"不看下一章会死"的悬念。
