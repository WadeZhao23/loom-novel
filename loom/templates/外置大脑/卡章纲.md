# 卡章纲

> 一章一句话的脊柱。大纲师 Agent 写第 N 章时,查第 N 行知道这章要完成什么、上一章结尾的钩子是什么。
> 这是跨章剧情连续性的主线(v0.1 跨章记忆靠它 + 上一章正文)。
> 你 learn 一章后,Loom 会在该章行下自动补一个缩进的「- [AI回顾] 摘要/伏笔」子块(这章实际写成了什么、埋/推/收了哪些伏笔),帮后面几十章接得住前情。它可手改可删,改不改都不影响你手写的规划行。
> 写法示范:「- 第1章:主角在最糟处境登场,金手指初现,章末抛一个让人想翻下一章的钩子」——每行=这章完成什么+章末钩子,前三章最好有小爽点闭环。
> (占位示例,换成你自己的;也可以在建书后用「AI 铺设定底稿」按书名起草前 5 章。)
